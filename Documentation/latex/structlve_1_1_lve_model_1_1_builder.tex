Charge un modèle à partir d\textquotesingle{}un fichier au format OBJ. 

Cette fonction charge un modèle à partir d\textquotesingle{}un fichier au format OBJ. Elle utilise la bibliothèque tinyobjloader pour lire le fichier et extraire les attributs des vertices tels que la position, la couleur, la normale et les coordonnées de texture. Les vertices sont stockés dans le vecteur \textquotesingle{}vertices\textquotesingle{} de l\textquotesingle{}objet \doxylink{structlve_1_1_lve_model_1_1_builder}{Lve\+Model\+::\+Builder}, tandis que les indices des vertices sont stockés dans le vecteur \textquotesingle{}indices\textquotesingle{}. Les vertices sont comparés pour détecter les doublons et les indices sont ajustés en conséquence pour assurer l\textquotesingle{}unicité des vertices.
