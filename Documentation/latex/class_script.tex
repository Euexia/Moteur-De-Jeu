Classe \doxylink{class_script}{Script}. 

Cette classe représente un script de base utilisé dans le moteur de jeu. Elle fournit des méthodes pour accéder aux différents modules du moteur, tels que le module de temps, le module de fenêtre, le module d\textquotesingle{}interface de rendu et le module d\textquotesingle{}entrée. Elle hérite de la classe de base \doxylink{class_i_script}{IScript}.
