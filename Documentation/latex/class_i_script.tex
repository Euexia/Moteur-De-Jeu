Interface de base pour les scripts. 

Cette classe définit une interface de base pour tous les scripts utilisés dans le moteur de jeu. Les scripts peuvent être initialisés, démarrés, mis à jour et détruits. Ils peuvent également avoir un nom et un type par défaut.
