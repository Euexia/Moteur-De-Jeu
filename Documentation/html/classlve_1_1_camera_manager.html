Description dÃ©taillÃ©e

Gère les caméras dans le moteur de jeu.

Cette classe gère les différentes caméras utilisées dans le moteur de jeu. Elle permet de créer, de supprimer et de manipuler les caméras, ainsi que de gérer les vues et projections associées. La classe CameraManager est une classe finale, ce qui signifie qu'elle ne peut pas être dérivée.
