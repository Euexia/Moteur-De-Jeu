Charge un modèle à partir d'un fichier au format OBJ.

Cette fonction charge un modèle à partir d'un fichier au format OBJ. Elle utilise la bibliothèque tinyobjloader pour lire le fichier et extraire les attributs des vertices tels que la position, la couleur, la normale et les coordonnées de texture. Les vertices sont stockés dans le vecteur 'vertices' de l'objet LveModel::Builder, tandis que les indices des vertices sont stockés dans le vecteur 'indices'. Les vertices sont comparés pour détecter les doublons et les indices sont ajustés en conséquence pour assurer l'unicité des vertices.
